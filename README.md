# Quiz App 🎯

A simple **Java-based Quiz Application** that allows users to answer multiple-choice questions and view their final score. This project is built using **Java and Maven** and is ideal for beginners learning Java application development.

## 📌 Features

- Multiple-choice quiz questions  
- Score calculation at the end of the quiz  
- Simple and easy-to-understand code structure  
- Maven-based project setup  

## 🛠️ Technologies Used

- Java  
- Maven  
- JavaFX (if applicable)  

## 📂 Project Structure

Quiz-app/
├── src/
│ ├── main/
│ │ ├── java/ # Java source files
│ │ └── resources/ # Application resources
├── target/ # Compiled output
├── pom.xml # Maven configuration file
└── README.md # Project documentation


## 🚀 How to Run the Project

### 1️⃣ Clone the Repository
git clone https://github.com/Abinaya7247/Quiz-app.git

cd Quiz-app


### 2️⃣ Make Sure Maven Is Installed

Check Maven installation:
mvn -version


### 3️⃣ Build the Project

mvn clean install

### 4️⃣ Run the Application

mvn javafx:run


*(If JavaFX is not used, run the main class directly from your IDE.)*

## 🧪 How to Use

1. Start the application  
2. Answer the quiz questions displayed  
3. Submit your answers  
4. View your final score
